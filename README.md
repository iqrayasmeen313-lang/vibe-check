# Vibe Checking (First JS App)

A fun interactive quiz that asks 5 lifestyle questions and reveals your vibe type based on the score.

## What it does

Users answer 5 multiple-choice questions. Each answer adds points to a score, and when the quiz ends, the app shows a result card with:
- emoji
- vibe title (e.g. "The Cozy Sloth", "The Galaxy Brain")
- description text
## How to run locally

1. Clone or copy the project folder.
2. Open the folder in your code editor (e.g. VS Code).
3. Open `index.html` directly in your browser (double-click or right-click open with browser).

Optional (recommended for local dev HTTP server):

- Using VS Code Live Server extension:
  - Install Live Server.
  - Right-click `index.html` > "Open with Live Server".
